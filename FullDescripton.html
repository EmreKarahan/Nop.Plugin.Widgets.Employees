This a payment processor plugin for the Icelandic payment processor Netgíro.
The plugin supports payments only (not voids, refunds, recurring payments or capture)

Features:

  • Simple configuration
  • Quick installatoin
  • Customers are redirected to Netgíró secure website to complete the payment and returned to your website upon successful payment

Installation instructions:

  1. Download the plugin archive.
  2. Go to admin area > configuration > local plugins.
  3. Upload the plugin archive using the "Upload plugin or theme" plugin.
  4. Scroll down through the list of plugins to find the newly installed plugin. And click on the "Install" button to install the plugin.
